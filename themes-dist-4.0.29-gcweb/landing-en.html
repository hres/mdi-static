Data extracts from the Medical Device System incident database

(ZIP Version - 132.3 MB)

The data set is updated 4 times per year and currently covers the following time period: 1977 to 2018-11-30, and includes all incidents as well as those recalls that are verified as acceptable by a Health Canada Inspector.

The data extract is a series of compressed UTF-8 text files extracted from the Medical Devices System. It is intended for users who are familiar with database structures and setting up their own queries. Get details about the data structure required to download the data. In order to use the data, the file must be loaded into an existing database or information system provided by the user.
